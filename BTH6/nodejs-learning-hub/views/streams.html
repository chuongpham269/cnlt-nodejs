Tổng hợp Streams (/streams)

🏠 Quay lại Trang chủ

4.1 Readable Stream (Hiển thị text từ file `story.txt`)

4.2 Writable Stream (Ghi file log)

Nhập liệu bên dưới, dữ liệu sẽ được push sang server lưu vào `log.txt` thông qua createWriteStream.

4.3 Transform Stream (Biến đổi chữ)

Nhập chữ thường bất kỳ. Server nhận stream, đưa vào Transform để chuyển thành IN HOA trước khi trả về.

4.4 Duplex Stream (Echo Message)

Gửi dữ liệu, Duplex stream server sẽ vừa nhận `_write` vừa trả `_read` cùng lúc chữ 'Echo:'.
